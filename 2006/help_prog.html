General Community Programs Information
Program Registration allows patrons to view programs offered by the library. Patrons may retrieve programs in either a calendar or browse view, depending on the initiated search. Program Registration is a useful way to familiarize yourself with the community programs offered by the library and allow the library to play an integral role in planning your life.
Calendar Navigation
The calendar offers a visual monthly view of the programs offered by the library.
 Month Hover
Hovering over the month in the first column will allow the calendar view to change between months
 Year Dropdown
The Year dropdown allows the user the ability to change the default calendar year in respects to the months.
 Program Type Dropdown
The Program Type Dropdown will allow users to change the default calendar view to display only programs of a specific type.
 Day Scroll
The display of programs per day may exceed the available fixed screen of the calendar, therefore a day scroll has been implemented to allow you to see all programs offered for a specific day.
 Program Information
Clicking on a program will display a pop up box with information about the specific program and it's registration status.
 Registration
The Program Information pop up box offers the option to Register. Selecting the register option with ask for patron verification, if you are not currently logged into a session.
Program Registration
After viewing the calendar or using advanced keyword searches for programs, you may select a program for registration. Upon selecting the register option, your patron account will be checked for patron blocks and eligibility. The system will auto-populate the email address in the form if you have an email address associated with your patron account. The email address on the form is editable and once registration is complete, a registration confirmation receipt will be sent to that email address.
My Programs
To see what programs you are currently registered for requires accessing your My Millennium account. Upon verification of your account a "x programs currently registered" link will display. Selecting this link will display currently registered programs. You are able to sort the view by Program Name or Date and cancel registration.
